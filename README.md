# Cris Clamis — Portfolio

A single-file, dark-themed developer portfolio built with HTML, CSS, and vanilla JavaScript.

## Features
- Typing-effect hero section
- Animated skill bars
- Project showcase (Learning Management System, MigoHealth, FarmNect)
- Working contact form (powered by Formspree)
- Downloadable resume
- Dark/light mode toggle
- Fully responsive
